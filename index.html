Task

1. Write the deepEqual function to check two objects for identity. 

          deepEqual({name: 'test'}, {name: 'test'}) // output true
          deepEqual({name: 'test'}, {name: 'test1'}) // output false
          deepEqual({name: 'test', data: {value: 1}}, {name: 'test', data: {value: 2}}) // output false
          deepEqual({name: 'test'}, {name: 'test', age: 10}) // false

solution - deepEqual

2. Write the chunkArray generator function, which returns an iterator that returns parts of the array of the specified length. 

          const iterator = chunkArray([1,2,3,4,5,6,7,8], 3);
          iterator.next() // { value: [1,2,3], done: false }
          iterator.next() // { value: [4,5,6], done: false }
          iterator.next() // { value: [7,8], done: false }
          iterator.next() // { value: undefined, done: true }

solution - chunkArray

3. Write a wrapper function that accepts an array of functions and their parameters as input, and returns an array of the results of their execution. The number of arguments of the executable function is not limited! 

          const f1 = (cb) => {cb(1)}
          const f2 = (a, cb) => {cb(a)}
          const f3 = (a, b, cb) => {setTimeout(() => cb([a, b]), 1000)}

          bulkRun(
            [
              [f1, []],
              [f2, [2]]
              [f3, [3, 4]]
            ]
          ).then(console.log)
          // Output: [1, 2, [3, 4]]

solution - bulkRun
